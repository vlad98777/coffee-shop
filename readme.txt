Coffee Shop Management System

Cистема управления сетью кофеен** с модульной архитектурой и полным циклом CRUD операций.

 Архитектура
- **Модели:** POJO сущности (CoffeeShop, Customer, Order...)
- **DAO:** Data Access Layer для работы с БД  
- **Сервисы:** Бизнес-логика и валидация
- **UI:** Консольные интерфейсы (31 функция)

Поддерживаемые БД
- **H2:** Для разработки и тестирования
- **Oracle:** Для production (скрипты в `/sql-scripts/`)
 Быстрый старт
```bash
cd coffee-shop-java
java -cp "classes;lib/h2.jar" com.coffeeshop.Main

 Модули системы
 Управление кофейнями (10+ функций)

 CRM клиентов (8+ функций)

 Обработка заказов (8+ функций)

 Каталог меню (4+ функций)

 Расписание сотрудников (5+ функций)

 Отчеты и аналитика (5+ функций)


 Технологии
- Java 11+
- H2 Database
- JDBC
- Паттерны: MVC, DAO, Service Layer
